# 🎓 Ignitext — Premium CGPA Calculator Web App

A modern, responsive web application for calculating CGPA (Cumulative Grade Point Average) with an elegant user interface and real-time calculations.

## ✨ Features

### Core Functionality
- **Dynamic Input Management**: Add/remove subjects with ease
- **Real-time CGPA Calculation**: Instant calculation as you input grades and credits
- **Grade to GPA Conversion**: Automatic conversion of letter grades to grade points
- **Multiple Grade Systems**: Support for various grading scales
- **Credits/Units Support**: Calculate weighted GPA based on course credits

### User Interface
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Styling**: Clean, gradient-based design using Tailwind CSS
- **Intuitive Controls**: Easy-to-use buttons and input fields
- **Visual Feedback**: Real-time updates and helpful UI indicators
- **Data Persistence**: Saves your input locally (browser storage)

### Extra Features
- 📊 **Result Summary**: Displays final CGPA with detailed statistics
- 🎨 **Premium Design**: Modern gradient backgrounds and smooth animations
- ♿ **Accessible**: Keyboard navigation and screen reader support
- 📱 **Mobile-Friendly**: Optimized touch interactions for mobile users
- 🔄 **Quick Reset**: Clear all data and start fresh with one click

## 🚀 Getting Started

### Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or build process required

### Installation
1. Download or clone the project
2. Open `index.html` in your web browser
3. Start calculating your CGPA!

## 📖 How to Use

1. **Add Subjects**: Click the "Add Subject" button to add a new course
2. **Enter Details**: 
   - Input the subject/course name
   - Select or enter the grade received (A, B, C, D, etc.)
   - Enter the credit hours for the course
3. **View Results**: Your CGPA updates automatically as you enter data
4. **Manage Courses**: 
   - Edit any subject by modifying the input fields
   - Remove subjects using the delete button
5. **Reset**: Click "Reset All" to clear everything and start over

## 📊 Grading Scale

The calculator uses the following standard grading scale:
- **A**: 4.0 points
- **B**: 3.0 points
- **C**: 2.0 points
- **D**: 1.0 points
- **F**: 0.0 points

## 💾 Data Storage

Your data is saved locally in your browser using localStorage. This means:
- ✅ Your data persists between sessions
- ✅ No data is sent to any server
- ✅ Your information remains private and secure
- ⚠️ Clearing browser data will clear saved information

## 🐛 Troubleshooting

**Issue**: Data not saving between sessions
- **Solution**: Check if your browser allows localStorage. Enable cookies/storage in browser settings.

**Issue**: Grades not calculating correctly
- **Solution**: Ensure you've entered valid grade letters (A, B, C, D, F) and numeric credit values.

**Issue**: App looks different on mobile
- **Solution**: The app is responsive. Try rotating your device or maximizing the browser window.
